① 实现点击元素变色效果

一开始的要求就是使用 data 的数组来做数据然后渲染列表并实现点击某个元素时，某个元素变色。

仔细思考一下，将过程拆分开。

  1. 首先是使用 v-for 指令循环生成元素。
  2. 然后考虑给每个元素绑定 click 事件。
  3. 点击时给当前点击元素添加指定类名同时移除其他所有元素的类名。

然后就开始蒙圈了，应该怎么给数组数据和类名动态对应呢？
纠结了半天，用了对象的方法。

<style>
  .red {
    color: red;
  }
</style>
<div id="app">
  <ul>
    <li 
    :key="index" 
    v-for="(value,key, index) in list" 
    :class="value?'red':''" 
    @click="handle(key)">{{ key }}</li>
  </ul>
</div>
<script>
  var vm = new Vue({
    el: '#app',
    data: {
      list:{
        'aaaaaaaaaa':true,
        'bbbbbbbbbbbb':false,
        'cccccccccccccc':false,
        'dddddddddddddddd':false,
        'eeeeeeeeeeeeeeeeee':false,
      }
    },
    methods: {
      handle(index){
        let list = this.list;
        for(let i in list){
          list[i]=false;
        }
        list[index]=true;
      }
    }
  })
</script>

然后兜兜转转半天看到别人的解决方法，瞬间感觉自己是个 ** 。

<style>
  .red {
    color: red;
  }
</style>
<div id="app">
  <ul>
    <li 
    :key="index" 
    v-for="(item, index) in list" 
    :class="{'red':currentIndex==index}" 
    @click="handle(index)">{{ item }}</li>
  </ul>
</div>
<script>
  var vm = new Vue({
    el: '#app',
    data: {
      list:[
        'aaaaaaaaaa',
        'bbbbbbbbbbbb',
        'cccccccccccccc',
        'dddddddddddddddd',
        'eeeeeeeeeeeeeeeeee',
      ],
      currentIndex: 0,
    },
    methods: {
      handle(index){
        this.currentIndex=index;
      }
    }
  })
</script>

只需要一个记录当前应该高亮的索引值，渲染元素时作为类名是否添加的判断条件，然后点击事件中动态修改这个索引值就可以。

② 动态切换渲染的底层原理

先给出一个小例子


例子很简单，就是根据 data 的一个 flag 值动态切换元素。

<div id="app">
  <div v-if="flag">
    <label for="use-phone">用户手机号</label>
    <input type="text" id="use-phone">
  </div>
  <div v-else>
    <label for="use-email">用户邮箱</label>
    <input type="text" id="use-email">
  </div>
  <button @click="flag=!flag;">切换类型</button>
</div>
<script>
  var vm = new Vue({
    el: '#app',
    data: {
      flag: true,
    }
  })
</script>

但是在切换的时候会注意到，输入的 value 值在元素切换的时候被保留了。
原因是 vue 底层的虚拟 dom 在更新时会比对新旧元素的改变，尽可能的减少修改的范围。
新旧元素除了 id 值以外都是相同的。因此， input 组件的 value 值被保留。
专业术语应该是 input 和 label 被复用。

③ 购物车实例

先给出演示

一个很简单的购物车例子，综合运用了 v-for v-if v-else v-bind v-on computed methods 等指令。

table {
  border: 1px solid #e9e9e9;
  border-collapse: collapse;
  border-spacing: 0;
}
th, td {
  padding: 8px 10px;
  border: 1px solid #e9e9e9;
  text-align: left;
}
th {
  background: #f7f7f7;
  color: #5c6b77;
  font-weight: 600;
}
<body>
<div id="app">
  <table v-if="books.length">
    <thead>
      <tr>
        <th></th>
        <th>书籍名称</th>
        <th>出版日期</th>
        <th>价格</th>
        <th>数量</th>
        <th>操作</th>
      </tr>
    </thead>
    <tbody>
      <tr v-for="(item, index) in books"
          :key="item.id">
        <td>{{ item.id }}</td>
        <td>{{ item.name }}</td>
        <td>{{ item.date }}</td>
        <td>{{ item.price | showPrice }}</td>
        <td>
          <button @click="decrement(index)" :disabled="item.count<=1">-</button>
          {{ item.count }}
          <button @click="increment(index)">+</button>
        </td>
        <td><button @click="remove(index)">移除</button></td>
      </tr>
    </tbody>
  </table>
  <div v-else>
    <h1>购物车为空</h1>
  </div>
  <h2>总价格 {{ totalPrice }}</h2>
</div>
var vm = new Vue({
  el: '#app',
  data: {  
    books: [
      {
        id: 1,
        name: '《算法导论》',
        date: '2006-9',
        price: 85.00,
        count: 1
      },
      {
        id: 2,
        name: '《UNIX编程艺术》',
        date: '2006-2',
        price: 59.00,
        count: 1
      },
      {
        id: 3,
        name: '《编程珠玑》',
        date: '2008-10',
        price: 39.00,
        count: 1
      },
      {
        id: 4,
        name: '《代码大全》',
        date: '2006-3',
        price: 125.00,
        count: 1
      },
    ]
  },
  methods: {        
    increment(index){
      this.books[index].count++;
    },
    decrement(index){
      this.books[index].count--;
    },
    remove(index){
      this.books.splice(index,1);
    }
  },
  filters: {
    showPrice(price){
      return '￥'+ price.toFixed(2);
    }
  },
  computed: {
    totalPrice(){
      let sum = 0;
      for(let i = 0; i < this.books.length; i++){
        sum+=this.books[i].price * this.books[i].count;
      }
      return sum;
    }
  }
})

通过指令来实现各种操作，最终都将改变体现到 data 中的 books 数组，页面内容也会相应改变。
在对价格渲染时，因为数据的原始性，我们需要对数据进行一定的修改，因为这种修改在页面中存在多处，所以写了一个 filter 过滤器来实现。

在对于总价格的累加处，有很多实现方式，列举一下。

  1. 原始的 for 循环，遍历数组，计算每一项对应的总价格并累加。
  2. 使用 for-in 循环
    let totalPrice = 0;
for(let i in this.books){
  const book = this.books[i];
  totalPrice += book.price * books.count;
}
return totalPrice;
  3. 使用 for-of 循环
    let totalPrice = 0;
for(let item of this.books){
  totalPrice += item.price * item.count;
}
return totalPrice;

④ Vue 双向绑定原理

vue 中的 v-model 指令实现了数据的双向绑定。我们可以通过另外两个指令来实现双向绑定。

<div id="app">
  <input type="text" v-model="message">
  {{message}}
</div>
<script src="./vue.js"></script>
<script>
  const app = new Vue({
    el: '#app',
    data: {
      message: 'a'
    }
  })
</script>

v-on + v-bind 配合产生数据双向绑定效果。

<div id="app">
  <input type="text" :value="message" @input="valueChange">
  {{message}}
</div>
<script src="./vue.js"></script>
<script>
  const app = new Vue({
    el: '#app',
    data: {
      message: 'a'
    },
    methods: {
      valueChange(e){
        this.message = e.target.value;
      }
    }
</script>

⑤ Vue 响应式注意点

vue 实例中 data 中的数组响应式存在于七个方法中， push pop splice shift unshift sort reverse ，除此之外可以使用 vue 官方提供的 set delete 方法来响应式更新数组或对象。

Vue.set(state.info, 'keyOrIndex', 'value')
Vue.delete(state.info, 'keyOrIndex')
